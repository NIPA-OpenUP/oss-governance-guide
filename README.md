# 기업 공개 소프트웨어 거버넌스 가이드 저장소

이 Repository는 기업 공개 소프트웨어 거버넌스 가이드를 작성하기 위한 공간입니다. 

기업 공개 소프트웨어 거버넌스 가이드는 정보통신산업진흥원(NIPA)이 주관하고 [오픈업](https://www.oss.kr/open_up_intro)이 연구를 수행하여 국내 기업의 올바른 오픈소스 활용을 안내하고자 작성하고 있습니다.
